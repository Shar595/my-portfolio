Project Overview

ResumeKeeper is a full-stack web application designed to help users create, edit, and maintain professional resumes online. The application integrates secure authentication, cloud data storage, and AI-powered content generation to enhance the resume-building experience.

Purpose

The goal of ResumeKeeper was to build a secure, scalable platform that allows users to manage their professional information while leveraging AI tools to generate polished resume content efficiently.

Key Features

  • Firebase Authentication for secure user login
  • Firestore database integration for resume data storage
  • AI-generated "About Me" and job descriptions
  • Project upload and AI-powered project summaries
  • Cloud Storage integration for document management

Technologies Used

  • Next.js
  • React
  • TypeScript
  • Firebase (Authentication, Firestore, Storage)
  • Gemini API
  • Tailwind CSS

Engineering Growth & Reflection

Building ResumeKeeper strengthened my understanding of full-stack development, cloud architecture, authentication systems, and AI integration. What initially felt overwhelming became a major learning breakthrough in debugging, deployment, and managing real-world web application workflows.
