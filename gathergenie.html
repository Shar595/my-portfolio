Project Overview

GatherGenie is an AI-powered web application designed to assist users in planning events with intelligent recommendations. The system generates suggestions for themes, decorations, food planning, and organization to simplify the event coordination process.

Purpose

The goal of GatherGenie was to explore AI-assisted user interaction and dynamic content generation while building a responsive and user-friendly web interface. The application demonstrates integration between front-end design and AI logic.

Key Features

  • AI-generated party themes and ideas
  • Smart event planning suggestions
  • Interactive user input handling
  • Responsive layout design

Technologies Used

  • Next.js
  • React
  • TypeScript
  • AI API Integration
  • Firebase (Development Environment)

Engineering Growth & Reflection

Developing GatherGenie enhanced my understanding of AI-driven applications, user experience design, and interactive system development. This project strengthened my ability to connect intelligent logic with clean interface implementation.
